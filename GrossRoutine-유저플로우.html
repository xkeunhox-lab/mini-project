GrossRoutine 유저 플로우

루틴을 등록해두는 루틴 관리와, 그 루틴을 불러와 실제로 기록하는 기록, 두 흐름이 어떻게 연결되는지 한 장으로 정리했습니다. 회의·개발 논의 시 화면 흐름 공통 기준으로 사용하세요.

📋 루틴을 만들고 다듬는 흐름

운동일마다 새로 입력하지 않도록, 반복해서 쓸 루틴을 미리 등록·수정해두는 화면입니다.

루틴 목록

등록된 루틴이 색상별 카드로 표시됩니다. 같은 색이 캘린더 배지·불러오기 목록에서도 동일하게 유지됩니다.

↓

➕ 루틴 추가 카드 클릭

빈 루틴 생성 모달이 열립니다.

기존 루틴 카드 클릭

루틴 상세 팝업 → 운동 항목 클릭 시 바로 수정 모드로 진입 (중간 상세 화면 없음)

↓

운동 정보 입력 / 수정

운동명(자동완성) · 세트 수 · 세트별 무게·횟수 입력. 세트 수를 바꿔도 이미 입력한 값은 유지됩니다.

↓

저장

루틴명 + 운동 1개 이상 확인 후 저장, 목록으로 복귀

🗑 루틴/운동 삭제

확인창 → 승인 시에만 삭제

기록 탭에서는 여기서 만든 루틴을 불러와 사용합니다 ↓
Tab · 기록

📅 캘린더에서 실제로 기록하는 흐름

날짜를 고르고, 루틴을 불러오거나 직접 운동을 추가해 그날의 운동을 기록합니다.

캘린더 화면

기록이 있는 날은 ✓, 루틴을 불러온 날은 루틴 색상 배지, 그 아래엔 그날의 총 볼륨(모든 세트 무게 합산)이 표시됩니다.

↓

배지의 ✕ 클릭 → 확인창 → 그 날짜 기록 즉시 삭제

날짜 화면에 들어가지 않고 캘린더에서 바로 처리되는 예외 경로

↓

날짜 클릭 → "오늘의 운동" 화면 진입

여기서부터는 아래 임시 편집본 영역 안에서 모든 조작이 이뤄집니다.

↓
🗂 임시 편집본 (Draft)
이 영역 안의 모든 조작은 화면에만 반영됩니다. 실제 데이터는 아래 저장을 눌러야만 바뀌고, 돌아가기를 누르면 지금까지의 변경은 전부 버려집니다.

📂 루틴 불러오기

루틴 선택 → 모든 운동을 한번에 표시. 운동 목록은 항상 루틴의 최신 정의를 따르고, 무게/횟수 값만 최근 기록이 있으면 우선 반영

🕐 이전 기록 불러오기

실제로 기록했던 과거 날짜 목록(오늘 제외)에서 하루 선택 → 그날의 운동을 통째로 복사. 완료 체크는 항상 초기화

➕ 운동 추가

운동명·세트수·세트별 무게/횟수 입력 후 추가

✏️ 세트 직접 수정

완료 체크박스, 세트별 + / − 버튼으로 개별 추가·삭제, 무게/횟수 직접 입력, 운동 삭제(확인창)

↓
